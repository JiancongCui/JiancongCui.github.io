Tips

Input

  1. Give AI a role.

    • ✅ You are an academic journal editor. Rewrite this abstract to meet publishing standards and improve clarity.

  2. Given AI Enough Context

    • ❌ Summarize this paper.
    • ✅ Summarize this neuroscience paper in 200 words, highlighting the main challenges, hypotheses, methods, and conclusion for a graduate-level audience.

  3. No “Please” or “Thanks”. This may change the output tone.

  4. Giving Examples to AI

    • Read this and say read when done: …
    • Using what you just read … write …

Thought Process

  1. Writing down steps directly.
    • ✅
      1. Summarize the article.
      2. Now revise it for an academic conference abstract, 250 words.
      3. Simplify the language slightly for non-specialists in the audience.

Output

  1. Give AI a target audience.

    • ❌ Explain quantum entanglement.
    • ✅ Explain quantum entanglement to a high school student using analogies and simple language.

  2. Give AI a specific format for the response.

    • ❌ Write about the impacts of climate change.
    • ✅ Create a table comparing the effects of climate change on agriculture in three countries, using peer-reviewed data only.

  3. Constraints

    • ✅ Write a … Avoid background information or future directions.
    • ✅ Write a … focusing only on the methodology.

  4. Setting the right tone with AI

    • ✅ Rewrite this paragraph in academic tone using active voice and hedging statements like ‘it appears’ or ‘the data suggests’."
